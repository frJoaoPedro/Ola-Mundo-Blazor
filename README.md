🚀 Olá Mundo - Blazor
Este repositório contém uma aplicação simples de "Olá Mundo" desenvolvida em Blazor, demonstrando a estrutura básica de componentes e a sintaxe Razor.

📋 Sobre o Projeto
O objetivo deste projeto é apresentar os conceitos iniciais do framework da Microsoft para web, focando em:

Estrutura de pastas de um projeto ASP.NET Core.

Criação e renderização de componentes .razor.

Data binding simples.

🛠️ Tecnologias Utilizadas
C# (.NET 8.0 ou superior)

Blazor WebAssembly (ou Server)

VS Code ou Visual Studio

🏗️ Estrutura Principal
No Blazor, tudo gira em torno de Componentes. O arquivo principal deste exemplo costuma ser o Index.razor (ou Home.razor), localizado na pasta Pages.
